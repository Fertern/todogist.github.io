# ToDoGist task manager app (Pure JavaScript)

## Technical task for "Ciklum University: JS Band Internship"

## In this app you can:

- Make task cards with title, description and priority
- Edit any existing card
- Change done/undone card status
- Sort cards by:
  +Title
  +Status
  +Priority
